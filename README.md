# google-ads-open-cli

A Google Ads CLI designed for AI agents. Wraps the official Google Ads API with simple, agent-friendly commands.

**Works with:** OpenClaw, Claude Code, Cursor, Codex, and any agent that can run shell commands.

> **Status: Under Development** — This project is not yet functional. Star/watch for updates.

## API Reference

- Official docs: https://developers.google.com/google-ads/api/docs/start
